# 英語 並び替えトレーナー（Real Grammar Unit 1-15）

英語教科書 **Real Grammar for Creative Communication（Unit 1〜15）** の英文を、**語句をタップして並べ替えて完成させる**学習Webアプリです。スマホ利用を想定したモバイルファースト設計。（参考: 同梱の「ドイツ語文法トレーナー」の構成を踏襲。）

## 概要・目的

- 元教材のスキャン（`英語-04〜33` の EXERCISES）と**授業スライド**（`IMG_5433〜5640`）から抽出した問題を、すべて**並び替え問題**に再構成。
- 解答は**スペル・文法を修正済み**（`英語_並び替え問題.md` と同内容）。
- 各 Unit の**文法解説**・**ヒント**・**覚え方のコツ**付きで、つまずかず学べる。
- 進捗（最高正答率）は端末内（localStorage）に保存。

## 機能

- **ホーム**：Unit 1〜15 をカード表示。各カードに学習テーマ・問題数・進捗バー。
- **文法解説**：各 Unit の要点・例文・表を表示。
- **並び替えクイズ**：日本語の指示を見て、下の語句タイルをタップして英文を組み立てる。
  - 解答エリアのタイルをタップで戻す／「↺ リセット」でやり直し。
  - 「答え合わせ」で即時判定。正解は紙吹雪、不正解は正しい語順を表示。
  - 一部の問題には**ダミー語句**（使わない語）が混ざる（授業スライド準拠）。
- **10問ずつ＆続きを解く**：全問を10問ごとに出題。終了後「続きを解く」で次へ。
- **解いている途中でも解説**：「📖 解説を見る」でいつでも文法解説を確認、閉じれば同じ問題に戻れる。
- **ヒント＆解説**：各問にヒント、回答後に詳しい解説＋「覚え方のコツ」。
- **結果画面**：正答率リング表示と、間違えた問題の復習リスト（あなたの解答／正解／解説）。

## ローカルで開く

`index.html` をブラウザでダブルクリックして開くだけで動作します（ビルド・サーバ不要）。

## GitHub Pages で公開する

1. このフォルダの内容をリポジトリのルートとして GitHub に push。
2. **Settings → Pages** を開く。
3. **Source** を `Deploy from a branch`、Branch を `main` / `(root)` に設定して保存。
4. 数分後 `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開。

※ `.nojekyll` を同梱しているため Jekyll の変換は行われません。

## 構成

```
英語並び替えサイト/
├── index.html          エントリ
├── .nojekyll           GitHub Pages の Jekyll 無効化
├── css/
│   └── style.css       スタイル（テーマ・タイルUI・アニメーション）
└── js/
    ├── app.js          ビュー管理・並び替えUI
    ├── quiz.js         並び替えクイズエンジン
    ├── storage.js      進捗保存（localStorage）
    └── data/
        ├── units.js    各Unitのメタ＋文法解説
        └── questions.js 全並び替え問題
```

## 技術

- 依存ライブラリ・ビルド不要の素の HTML / CSS / JavaScript。
- `prefers-color-scheme` でダークモード、`prefers-reduced-motion` でアニメーション抑制に対応。
- データ（解説・問題）とロジック（UI・エンジン）を分離。問題の追加・修正は `js/data/questions.js` を編集するだけ。
